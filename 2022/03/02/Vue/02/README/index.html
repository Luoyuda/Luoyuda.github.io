Vue 源码阅读（二） Vue 初始化

2022-03-03 源码阅读 Vue 0 评论 字数统计: 1.5k(字) 阅读时长: 7(分)

Vue 初始化

当我们创建一个Vue实例时

  1. 创建Vue类：构造函数、原型方法、静态方法
  2. 创建Vue实例：初始化数据、事件、模板

Vue 类

通过查看打包内容可知，Vue 构造函数在 src/platforms/web/entry-runtime.js 中

接着看 /src/platforms/web/runtime/index.js

这里实际上是对Vue构造函数进行一些基于原型上的扩展、平台代码、devtools代码的处理

接着进入 import Vue from 'core/index'

居然也不在这个文件中，接着往里面读 ./instance/index

  1. initMixin ：src/core/instance/init.js 添加了 _init

  1. stateMixin： src/core/instance/state.js 添加了 $data、$props 、$watch、$set 、$delete

  2. eventsMixin： src/core/instance/event.js 添加了 $on、$off 、$once、$emit

  1. lifecycleMixin： src/core/instance/lifecycle.js 添加了 _update、$forceUpdate 、$destroy
  1. renderMixin： src/core/instance/render.js 添加了 RenderHelpers 、$nextTick、_render

再回到 initGlobalAPI: src/core/global-api/index.js

以上就是 Vue 类中的基本内容

new Vue()

由上面可知，new Vue() 时构造函数执行的方法实际为 _init
